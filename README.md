# linguanova-translate-bot
A modern and vibrant web-based translation bot designed to provide quick and accurate translations across a wide range of global languages, including several Nigerian indigenous languages. Powered by Google's Gemini AI model, this bot offers a seamless translation experience directly in your browser.

## ‚ú® Features

-   Multi-Language Support: Translates between popular world languages like English, Spanish, French, German, Mandarin Chinese, Arabic, Hindi, Portuguese, Russian, Japanese, Korean.
-   Nigerian Language Inclusion: Special support for Yoruba, Hausa, Igbo, Tiv, and Edo (Bini) languages.
-   Dynamic Language Detection: Automatically detects the source language for convenient translation.
-   Intuitive User Interface: Clean, modern, and vibrant design with an animated background, smooth transitions, and clear input/output areas.
-   Real-time Feedback: Includes a loading spinner during translation and custom modals for user notifications and error handling.
-   Single File Deployment: All HTML, CSS (Tailwind CSS via CDN), and JavaScript are contained within one index.html file for easy setup and deployment.

## üöÄ How to Use

1.  Enter Text: Type or paste the text you wish to translate into the "Text to Translate" box.
2.  Select Languages:
    * Choose your "Source Language" from the dropdown. You can select "Detect Language" if you're unsure.
    * Choose your "Target Language" (the language you want to translate into).
3.  Translate: Click the "Translate" button.
4.  View Result: The translated text will appear in the "Translated Text" section below.
